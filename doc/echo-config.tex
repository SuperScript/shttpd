\section{The \cmd{echo-config} program}

\subsection{Interface}
\begin{code}
  echo-config \var{user} \var{loguser} \var{dir} \var{port}
\end{code}
where \cvar{user} and \cvar{loguser} are system account names,
\cvar{dir} is an absolute path name, and \cvar{port} is a TCP port.
The port may be specified as a name from \cmd{/etc/services} or a
numeric value.

\cmd{echo-config} creates the directory \cvar{dir} as an
\href{svscan}{http://cr.yp.to/daemontools/svscan.html}-format service
directory, configured to run \cmd{echo-httpd} on the TCP port
\cvar{port}, running \cmd{echo-httpd} as \cvar{user} and creating log
files as \cvar{loguser}.
