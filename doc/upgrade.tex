\subsection{0.52 to 0.53}
The \cmd{redir-httpd} program now logs the \cmd{Referer:} header in requests.
Modify your log analysis programs accordingly.

Redirection data for the host \cvar{HOST} is now located in
\cmd{ROOT/\var{HOST}/data.cdb}.
