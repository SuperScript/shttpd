\section{0.52 to 0.53}
The \cmd{redir-httpd} now logs the \cmd{Referer:} header in requests.  Modify
your log analysis programs accordingly.

Redirection data for the host \carg{host} is now located in
\cmd{ROOT/\arg{host}/data.cdb}.
